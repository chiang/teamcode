# Runner 시작하기

Runner 를 실행하기 전에 먼저 환경 설정이 필요합니다.

## 환경 설정하기

Runner 설정 파일은 TeamCode 홈 디렉터리 밑의 ``config/runner.yml`` 에서 찾을 수 있습니다.

설정 파일 내용은 아래와 같습니다.

```
token: 23filjafij2jfalafa
url: http://localhost
check-interval: 13s
```

| Key            | Value         | 설명   |
| -------------- |-------------| ----- |
| token          | Hash 문자열 | $1600 |
| url            | 팀코드 서버 주소      |   로컬 호스트로 지정하기 말고 반드시 도메인 주소를 지정하시기 바랍니다. |
| check-interval | are neat      |    $1 |

``token`` 값은 ㅁㄹ
